You should install
- Ubuntu Linux OS
- Server : NginX / MariaDB / PHP7.x-FPM
- Package : UNZIP
- Need Source: https://github.com/PHPOffice/PHPExcel (License : LGPL)
- FTP Client Uploader(e.g. Filezilla)

You can run : (in CLI or Web Browser)
0. Database Connection : data_connect.php
1. Data Schema : data_format.php
2. Data Upload : data_file.php
3. Data Save : data_input.php
4. Data View : data_heatmap.php
